# Virtusa Mini Projects

This repository contains mini projects developed using **Java, Python, and SQL**.

## 📂 Projects Included

### 1. Java – Library Management System
A console-based application to manage library operations efficiently.

**Features:**
- Add, remove, and update books  
- User registration system  
- Issue and return books  
- Fine calculation for late returns  
- Search books by title and author

**Concepts Used:**
- Object-Oriented Programming (OOP)  
- ArrayList  
- Date handling  

### 2. Python – Fare Calculator (FareCalc)
A ride fare estimation system with dynamic pricing.

**Features:**
- Distance-based fare calculation  
- Vehicle type pricing (Economy, Premium, SUV)  
- Surge pricing during peak hours  
- Error handling for invalid inputs
- Generates formatted ride receipt  

**Concepts Used:**
- Dictionaries  
- Functions  
- Conditional statements  
- Exception handling  

### 3. SQL – Retail Sales Analysis
A relational database system to analyze online retail sales.

**Tables:**
- Customers  
- Products  
- Orders  
- Order_Items  

**Key Queries:**
- Top-selling products  
- Most valuable customers  
- Monthly revenue  
- Category-wise sales  
- Inactive customers  

**Concepts Used:**
- SQL Joins  
- GROUP BY, SUM  
- Relational database design
